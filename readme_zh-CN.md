> :cl: 前端项目初始化命令行工具

## :books: 命令

|   名称   |        描述        |
| :------: | :----------------: |
|  `add`   |     添加新模板     |
| `delete` |      刪除模板      |
|  `list`  |    列出所有模板    |
|  `init`  | 根据模板初始化项目 |
|  `set`   |    修改配置文件    |
